# Java

Cookbook на Java 11+ `HttpClient` с аутентификацией по API-токену через `:8006`.

```bash
cd examples/java
javac Cookbook.java && java Cookbook
```

Сторонние JSON-библиотеки не нужны — ответы разбираются простыми строковыми
вспомогательными функциями, достаточными для лабораторного smoke-теста.
